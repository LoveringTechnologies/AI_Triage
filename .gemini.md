# 🏥 AI Emergency Triage System - Project Overview

## 📌 Project Status: v2.1.0 (Live & Clinical Grade)

A multi-platform clinical triage system utilizing a Hybrid Intelligence Engine (Random Forest ML + Expert Clinical Logic). The system categorizes patients into a 5-zone clinical model and supports real-time census tracking, patient discharge, and protocol-driven next actions.

## 🗂 File Structure
- `triage_api.py`: FastAPI Backend (v1.4.2) with Age-Stratified Logic.
- `TriageMobile/App.js`: React Native / Expo Mobile Suite with Census & Protocols.
- `generate_clinical_data.py`: High-fidelity clinical data generator (50k+ samples).
- `triage.db`: SQLite database for logs, users, and notifications.
- `Model Training Data/`: Version-controlled clinical datasets.

## 📊 Clinical Intelligence Layers

### 1. Age-Stratified CTAS Engine
- **PaedCTAS (12-18)**: Normalized thresholds for Teens (e.g., RR 30 is the critical threshold).
- **Adult CTAS (18+)**: Standard adult thresholds (e.g., RR 26 is the critical threshold).

### 2. Expert Protocols (Safety Floor)
- **Sepsis Alert**: Fever + High HR/RR + BP Drop.
- **Stroke Alert**: FAST Neuro Symptoms + Hypertension.
- **Shock Index**: Real-time `HR/SBP` tracking for hidden hypoperfusion.
- **Hypoglycemia**: Critical alerts for low glucose (<70 mg/dL).

### 3. Next-Action Protocols
Instantly provides clinical checklists to staff upon triage:
- *Resuscitation*: Stat MD Eval, Type & Cross, Continuous Monitoring.
- *Sepsis*: Fluid Bolus, Stat Antibiotics, Blood Cultures.
- *Stroke*: Code Stroke, Stat CT, POC Glucose.
